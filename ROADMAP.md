# Roadmap

Ориентир, не обязательство. Порядок отражает приоритет «сначала границы, потом удобства».

> Разбивка по спринтам (2-недельные, с задачами и Definition of Done) — в [`docs/SPRINTS.md`](docs/SPRINTS.md). Этот файл — крупные фазы; SPRINTS — единицы работы.

## Фаза 1 — Скелет с границами (текущая)

Цель: минимальный агент, у которого границы стоят раньше фич.

- [ ] Host-ядро: inbound/outbound очереди на SQLite, петля оркестратора
- [ ] Channel Adapter для Telegram (официальный Bot API), deny-by-default авторизация, pairing владельца
- [ ] Sandbox исполнения (Docker) с deny-all egress и явными mount'ами
- [ ] Credential Broker (локальный proxy) — агент не видит ключи
- [ ] Gate engine с градуированной ценой действия (read-only / обратимое / необратимое), fail-closed
- [ ] Audit log

## Фаза 2 — Память и обучение

- [ ] Эпизодическая память с полнотекстовым поиском (без LLM)
- [ ] Семантическая память: эпистемические статусы, provenance, evidence
- [ ] Promotion-гейт (corroborated автоматически, verified для необратимого)
- [ ] Детерминированная курация (staleness, usage, дедуп, decay)
- [ ] Snapshot/rollback перед мутациями хранилища

## Фаза 3 — Карантин входа

- [ ] Quarantine-плоскость: Q-LLM без инструментов
- [ ] Классификация «от владельца / недоверенное» по provenance
- [ ] Правило «недоверенные данные не инициируют tool-call в том же ходе»
- [ ] Обработка пересланного контента, веб-страниц, вложений

## Фаза 4 — Навыки и автоматизации

- [ ] Декларативные навыки (markdown + capability-манифест)
- [ ] Навыки с кодом — только в sandbox, через verifiable loop (dry-run → corroborated)
- [ ] Установка навыков из git с pinned-версией (без curl|bash, без runtime-install в хосте)
- [ ] Scheduler (cron как сообщения во входную очередь)

## Фаза 5 — Экономика и эксплуатация

- [ ] Budget engine: дневной лимит, приоритет интерактива над фоном
- [ ] Явная деградация при исчерпании бюджета + уведомление
- [ ] Метрика reuse_rate, отключение бесполезного self-improvement
- [ ] Документация по развёртыванию (self-hosted)
